"scavenge_player:consume_item"
Consumes the Held Item by the desired amount

"amount": The amount that should be Consumed
"chance": How likely that it is being consumed (Optional, Default: 100.0)
"jei_visible": If the JEI description should be shown (Optional, Default: false)
"jei_description": JEI Description that should be displayed (Optional, Default: scavenge_player.consume_item.desc)
"jei_simple": If the jei_description should be treated as a Simple String or a full ITextComponent JsonObject (Optional, Default: true)

Incompats:
[scavenge_player:damage_item]
