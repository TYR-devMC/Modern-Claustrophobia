"scavenge_player:set_nbt"
Adds/Removes NBTData from the Players Persistent data

"nbt": The NBTData that should be used
"remove": If the NBTData should be merged with or removed from the player
"jei_visible": If the JEI description should be shown (Optional, Default: false)
"jei_description": JEI Description that should be displayed (Optional, Default: scavenge_player.set_nbt.desc)
"jei_simple": If the jei_description should be treated as a Simple String or a full ITextComponent JsonObject (Optional, Default: true)
