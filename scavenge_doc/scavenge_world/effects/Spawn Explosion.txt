Spawns a Explosion

"flame": If the Explosion should spawn fire (Optional, Default: false)
"power": The Power of the Explosion (Optional, Default: 4.0)
"jei_visible": If the JEI description should be shown (Optional, Default: false)
"jei_description": JEI Description that should be displayed (Optional, Default: scavenge_world.spawn_explosion.desc)
"jei_simple": If the jei_description should be treated as a Simple String or a full ITextComponent JsonObject (Optional, Default: true)
